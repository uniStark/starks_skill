# starks-skill 项目记忆区

本目录由 `starks` skill 在每次任务完成后自动沉淀**项目总结与记忆索引**，供你在 Obsidian 内人工审查、跨项目互通。

- 这是有意保留的**正文内容区**（你长期阅读 / 拥有），不是临时 AI 脚手架，故不归入 `agents/`。
- 每个项目一个子文件夹：`<project>/summary.md`（总结）+ `<project>/memory.md`（记忆索引）。
- 跨项目互通靠 `_index.md` + `[[wikilink]]` + Obsidian graph。
- starks 写入时遵守 vault `AGENTS.md`：学你的笔风、写短、绝不碰 `private/`。
