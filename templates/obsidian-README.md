# starks-skill 项目记忆区

本目录供 `starks` 在用户对本次枚举内容明确授权后沉淀**项目总结与稳定事实**，供你在 Obsidian 内人工审查、跨项目互通；默认不读、不写。

- 这是有意保留的**正文内容区**（你长期阅读 / 拥有），不是临时 AI 脚手架，故不归入 `agents/`。
- 每个项目一个子文件夹：`<project>/summary.md`（总结）+ `<project>/memory.md`（记忆索引）。
- `_index.md` 与 Obsidian graph 只供人类导航；机器按 `summary.md` 的 `repo_id`、别名和关系路由。
- starks 写入时遵守 vault `AGENTS.md`：学你的笔风、写短、绝不碰 `private/`。
